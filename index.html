Integration Guide

Four modules to add to www.culturalmusings.com. Each is self-contained and drops into the existing nav/page structure. Use the tabs above to preview and configure each module.

STEP I

Add Raga Kavya Nav Buttons New

Copy the two new .mega-trigger buttons (Raga Kavya Creator + Raga Kavya) and the new drawer panel HTML into your existing nav. The drawer panel links directly to ragakavyacreator.culturalmusings.com and ragakavya.culturalmusings.com. See the Raga Kavya Nav tab for the exact code with copy buttons.

STEP II

Astro Predictor + Mantra Generator ⚡ Groq · llama-3.3-70b

Embeds as a new section or modal. Takes name, date and time of birth, generates 8 paragraphs of astrological interpretation + corresponding Vedic mantras. Now calls Groq API directly from the browser — no backend server needed. Includes full disclaimer + researcher note. Switch to the Astro Predictor tab to test it live.

STEP III

Dual Research Search ⚡ Groq · llama-3.3-70b

Two search modes: API mode (queries Groq directly to synthesise answers from the research domains) and Non-API / GitHub mode (links directly to github.com/ramachandra1975 and the curated subdomain index). No backend server required.

STEP IV

Dynamic Token Manager — Status Monitor

Monitors the Groq API key status and tracks usage. Since Groq handles auth via Bearer token in request headers (no separate refresh cycle needed), this panel now shows live key health and lets you swap the key if it gets rate-limited or revoked.
